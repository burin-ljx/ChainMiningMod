# Chain Mining | [连锁挖掘](README.cn.md)

This mod allows you to simultaneously mine surrounding blocks of the same type while mining blocks.

- Changelog
    - 1.0.0
        - Implemented the chain mining functionality
        - By default, hold the `~` key to enable chain mining, and the key can be modified
        - Maximum mining count is 64
        - No tool restrictions
        - Enabling chain mining does not reduce hunger value
